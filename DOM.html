Document Object Model (DOM)

Notes

The DOM (Document Object Model) is a structured representation of an HTML document. It allows JavaScript to dynamically update content, styles, and structure on a webpage.

The DOM represents an HTML document as a tree. Each element in HTML is a node in the tree.

Returns a collection (HTMLCollection) of elements with the given class. when selected by CLASS

Two commonly used collections of elements are HTMLCollection and NodeList.
